\doxysubsection{Detailed Description}
Represents a vertex (or settlement location) on the game board. 

The {\ttfamily \doxylink{class_vertex}{Vertex}} class is responsible for managing the state and appearance of a vertex on the board, where players can place settlements or cities. It tracks availability, ownership, and whether the vertex is highlighted.
