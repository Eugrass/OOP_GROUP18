\doxysubsection{Detailed Description}
Represents the start menu screen for the game. 

The {\ttfamily \doxylink{class_start_menu}{Start\+Menu}} class is the first screen players interact with when they launch the game. It provides buttons for the player to choose options like starting the game, adjusting player count, randomizing the board, and exiting the game. The class manages the visual display of these options and responds to user input.
