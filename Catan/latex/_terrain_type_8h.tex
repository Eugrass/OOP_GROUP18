Represents the types of terrain available on the game board. 

The {\ttfamily Terrain\+Type} enum defines the different types of terrain that can be found on the game board. Each terrain type is associated with a specific resource or gameplay effect\+:
\begin{DoxyItemize}
\item {\ttfamily FOREST}\+: Produces wood (lumber).
\item {\ttfamily HILLS}\+: Produces brick.
\item {\ttfamily MOUNTAINS}\+: Produces ore.
\item {\ttfamily FIELDS}\+: Produces grain.
\item {\ttfamily PASTURE}\+: Produces wool.
\item {\ttfamily DESERT}\+: Does not produce resources and is typically associated with the robber.
