\doxysubsection{Detailed Description}
Represents a player in the game, managing resources, cards, and actions. 

The {\ttfamily \doxylink{class_player}{Player}} class stores and manages the player\textquotesingle{}s resources, development cards, victory points, and other attributes such as the player\textquotesingle{}s color and ID. It provides methods for performing actions such as building roads and settlements, using development cards, and checking the player\textquotesingle{}s victory status.
