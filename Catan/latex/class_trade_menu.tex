\doxysubsection{Detailed Description}
Handles the trading interface and logic during a trade between players. 

The {\ttfamily \doxylink{class_trade_menu}{Trade\+Menu}} class provides the user interface for players to propose, accept, or decline trades. Players can offer and request resources, select other players to trade with, and interact with the trade menu.
