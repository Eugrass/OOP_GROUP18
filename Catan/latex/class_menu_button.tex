\doxysubsection{Detailed Description}
Represents a button in the menu system. 

The {\ttfamily \doxylink{class_menu_button}{Menu\+Button}} class extends the {\ttfamily \doxylink{class_button}{Button}} class to define buttons that are specifically used in the game\textquotesingle{}s menus. It provides constructors to set the button\textquotesingle{}s text, font, and position.
