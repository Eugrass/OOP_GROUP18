\doxysubsection{Detailed Description}
Represents a clickable button for resource selection in the game. 

The {\ttfamily \doxylink{class_resource_button}{Resource\+Button}} class provides the functionality needed to display a resource button on the screen, handle user interactions, and trigger actions when clicked. It uses an SFML sprite for visual representation and allows the button\textquotesingle{}s position and appearance to be customized.
