enum class TerrainType
strong

Represents the types of terrain available on the game board.

The TerrainType enum defines the different types of terrain that can be found on the game board. Each terrain type is associated with a specific resource or gameplay effect:

  • FOREST: Produces wood (lumber).
  • HILLS: Produces brick.
  • MOUNTAINS: Produces ore.
  • FIELDS: Produces grain.
  • PASTURE: Produces wool.
  • DESERT: Does not produce resources and is typically associated with the robber.
