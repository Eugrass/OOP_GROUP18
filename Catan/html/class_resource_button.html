Detailed Description

Represents a clickable button for resource selection in the game.

The ResourceButton class provides the functionality needed to display a resource button on the screen, handle user interactions, and trigger actions when clicked. It uses an SFML sprite for visual representation and allows the button's position and appearance to be customized.
